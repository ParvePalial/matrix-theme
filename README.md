## Installation

1. Install and Launch [Visual Studio Code](https://code.visualstudio.com/)
2. *Settings* > *Extensions* (or `Ctrl+Shift+X`)
3. Search for **Matrix Theme**
4. Click *Install*
5. *Settings* > *Themes* > *Color Theme* > *Matrix Theme*
